# 적외선 센서 기반 자율주행 프로젝트 구조

## 프로젝트 개요

적외선 센서를 활용하여 장애물 데이터를 수집하고, 주행 판단 알고리즘을 통해 회피 경로를 결정한 뒤, 모터 제어를 통해 자율주행을 구현하는 단계형 프로젝트로 구성한다. 이후 성능 검증 및 고도화를 통해 실제 자율주행 시스템 설계 역량을 강화한다.

이 프로젝트는 개인 프로젝트 수준에 맞춰 저가형 적외선 센서와 소형 주행 로봇 플랫폼을 사용하며, 자율주행의 범위는 장애물 감지 및 회피 주행에 한정한다.

## 4계층 구조

### 1계층. 센서 데이터 수집

적외선 센서를 활용해 전방, 좌측, 우측 장애물 데이터를 수집한다. 이 계층은 자율주행 시스템의 가장 기초가 되는 입력 계층으로, 센서에서 들어오는 원시 신호를 주행에 활용 가능한 형태로 정제하는 역할을 담당한다. 센서값 보정, 노이즈 제거, 임계값 설정을 거쳐 거리 또는 장애물 유무를 판별할 수 있는 기초 데이터를 생성한다.

**세부 구성**

- 입력: 전방, 좌측, 우측 적외선 센서의 아날로그 또는 디지털 값
- 처리: 센서값 읽기, 이동평균 필터 또는 평균값 보정, 임계값 비교
- 출력: `front_detected`, `left_detected`, `right_detected`와 같은 장애물 판별값

**구현 흐름**

1. 센서 초기화와 핀 매핑을 수행한다.
2. 일정 주기로 센서값을 읽어 온다.
3. 센서별 기준값을 적용해 잡음을 줄이고 보정한다.
4. 장애물 유무 또는 상대 거리 수준을 판정한다.
5. 판단 로직 계층으로 정제된 센서 상태를 전달한다.

**적용 예시**

- 전방 센서가 임계값 이하를 감지하면 전방 장애물 존재로 판정한다.
- 좌측 센서만 장애물을 감지하면 좌회전 제한 상태로 분류한다.
- 좌우 센서가 모두 장애물을 감지하지 않으면 직진 가능 상태로 전달한다.

**주요 구현 항목**

- 전방, 좌측, 우측 적외선 센서 데이터 수집
- 센서값 노이즈 제거 및 기초 보정
- 장애물 감지 임계값 설정
- 거리 추정 또는 장애물 유무 이진 판별

**주요 산출물**

- 센서 연결 회로 및 입출력 정의
- 센서값 수집 로그
- 임계값 기준표와 보정 결과
- 센서 상태 정의표
- 장애물 감지 조건서

### 2계층. 주행 판단 알고리즘

수집된 센서 데이터를 바탕으로 직진, 정지, 좌회전, 우회전, 후진 여부를 판단한다. 이 계층은 1계층에서 전달된 장애물 판별값을 해석해 주행 명령으로 변환하는 두뇌 역할을 담당한다. 장애물 위치와 우선순위에 따라 회피 로직을 설계하고, 단일 장애물과 복수 장애물 상황 모두에 대응할 수 있는 의사결정 규칙을 정의한다.

**세부 구성**

- 입력: 센서 계층에서 전달된 장애물 감지 상태와 거리 수준
- 처리: 조건 분기, 우선순위 비교, 회피 규칙 적용, 예외 상황 처리
- 출력: `GO_FORWARD`, `TURN_LEFT`, `TURN_RIGHT`, `STOP`, `REVERSE`와 같은 주행 명령

**구현 흐름**

1. 센서 상태를 받아 현재 주변 환경을 해석한다.
2. 전방 장애물 유무를 먼저 확인해 긴급 정지 여부를 결정한다.
3. 좌우 공간 여유를 비교해 회피 방향을 선택한다.
4. 복수 장애물 상황에서는 정지 또는 후진 우선 규칙을 적용한다.
5. 최종 주행 명령을 구동 제어 계층으로 전달한다.

**의사결정 예시**

- 전방 장애물이 없고 좌우도 비어 있으면 직진한다.
- 전방 장애물이 있고 좌측이 비어 있으면 좌회전한다.
- 전방 장애물이 있고 좌측이 막혀 있으며 우측이 비어 있으면 우회전한다.
- 전방, 좌측, 우측이 모두 막혀 있으면 정지 후 짧은 후진을 수행한다.

**주요 구현 항목**

- 직진, 정지, 좌회전, 우회전, 후진 판단 로직
- 좌우 장애물 우선순위 기반 회피 규칙
- 정면 장애물 감지 시 긴급 정지 또는 후진 로직
- 복수 장애물 발생 시 예외 처리 규칙

**주요 산출물**

- 상태 전이도 또는 의사결정 플로우차트
- 주행 판단 알고리즘 설명서
- 시나리오별 입력-출력 매핑표
- 조건문 또는 상태머신 설계서
- 장애물 상황별 회피 규칙표

### 3계층. 구동 제어 및 자율주행 실행

모터, 바퀴, 구동부를 제어하여 실제 주행 동작을 수행한다. 이 계층은 2계층에서 생성한 주행 명령을 실제 물리 동작으로 변환하는 실행 계층이다. 속도 제어, 방향 전환, 충돌 방지, 정지 제어를 구현하여 판단 결과가 실제 이동으로 이어지도록 한다.

**세부 구성**

- 입력: 판단 계층에서 전달된 주행 명령
- 처리: 모터 속도 설정, 회전 방향 제어, 주행 시간 제어, 정지 신호 처리
- 출력: 실제 차량 또는 로봇의 직진, 회전, 정지, 후진 동작

**구현 흐름**

1. 판단 계층에서 전달된 명령을 수신한다.
2. 명령 종류에 따라 좌우 모터의 속도와 방향을 설정한다.
3. 직진 또는 회전을 일정 시간 또는 조건까지 수행한다.
4. 추가 센서 입력이 위험 상황을 감지하면 즉시 정지한다.
5. 동작 결과를 로그로 기록해 검증 계층에서 활용할 수 있게 한다.

**제어 예시**

- `GO_FORWARD`: 좌우 모터를 동일 속도로 구동한다.
- `TURN_LEFT`: 우측 모터 속도를 높이거나 좌측 모터를 감속해 좌회전한다.
- `TURN_RIGHT`: 좌측 모터 속도를 높이거나 우측 모터를 감속해 우회전한다.
- `STOP`: 모든 모터 출력을 0으로 설정한다.
- `REVERSE`: 짧은 시간 후진 후 다시 판단 계층으로 복귀한다.

**주요 구현 항목**

- DC 모터 또는 서보 모터 제어
- 직진, 좌회전, 우회전, 후진 구동 제어
- 속도 조절 및 단계별 감속 제어
- 충돌 방지를 위한 즉시 정지 제어

**주요 산출물**

- 모터 제어 코드
- 주행 실행 시연 영상 또는 주행 로그
- 기능별 테스트 결과표
- 명령-동작 매핑표
- 속도 제어 파라미터 정리표

### 4계층. 성능 검증 및 고도화

주행 성공률, 장애물 회피 정확도, 반응속도를 측정한다. 이 계층은 앞선 1~3계층이 올바르게 연동되는지 확인하고, 센서 오차 보정, 주행 안정화, 복수 장애물 대응 등으로 시스템을 고도화하는 검증 계층이다. 실제 환경 적용 가능성을 높이기 위해 수집-판단-제어의 전체 흐름을 반복 점검한다.

**세부 구성**

- 입력: 센서 로그, 판단 결과, 주행 실행 기록, 테스트 시나리오 결과
- 처리: 성공률 분석, 오동작 원인 파악, 임계값 재조정, 회피 로직 개선
- 출력: 개선된 임계값, 수정된 알고리즘, 향상된 주행 성능

**구현 흐름**

1. 단일 장애물, 좌우 장애물, 복수 장애물 시나리오를 설계한다.
2. 각 시나리오에서 주행 성공 여부와 반응속도를 측정한다.
3. 실패 사례를 분석해 센서 임계값 또는 판단 규칙을 수정한다.
4. 수정 후 동일 조건에서 반복 테스트를 수행한다.
5. 개선 전후 성능을 비교해 최종 결과를 정리한다.

**고도화 예시**

- 센서 민감도가 너무 높아 오탐이 많으면 임계값을 재설정한다.
- 회전 반경이 과도하면 모터 속도 비율을 조정한다.
- 복수 장애물 상황에서 반복 정지가 발생하면 후진 후 재탐색 규칙을 추가한다.
- 주행 중 흔들림이 크면 감속 구간과 정지 조건을 세분화한다.

**주요 구현 항목**

- 단일 장애물 회피 성공률 측정
- 좌우 방향 전환 정확도 측정
- 연속 장애물 환경 반복 주행 테스트
- 센서 오차 및 노이즈 상황 안정성 점검

**주요 산출물**

- 성능평가 체크리스트
- 테스트 기록지와 개선 전후 비교표
- 고도화 개선안 및 최종 결과 보고서
- 시나리오별 성공률 통계표
- 최종 성능 요약서

## 레이어 간 연계 구조

본 프로젝트는 각 계층이 독립적으로 존재하는 것이 아니라, 아래와 같은 흐름으로 유기적으로 연결된다.

1. 1계층에서 장애물 감지 데이터를 생성한다.
2. 2계층에서 감지 데이터를 해석해 주행 방향과 행동을 결정한다.
3. 3계층에서 결정된 명령을 실제 모터 동작으로 실행한다.
4. 4계층에서 실행 결과를 검증하고, 다시 1~3계층의 파라미터와 로직을 개선한다.

즉, 본 프로젝트는 `센서 인식 -> 상황 판단 -> 물리 제어 -> 성능 개선`의 순환형 구조를 가지며, 최종적으로 안정적인 자율주행 구현을 목표로 한다.

## 3계층 축약 표현

필요 시 본 프로젝트는 아래 3계층 구조로도 요약할 수 있다.

1. 센서 인식
2. 판단 알고리즘
3. 주행 제어

## 테스트 및 검증 항목

- 단일 장애물 감지 후 정지 또는 회피가 정상 동작하는지 확인
- 좌우 장애물 위치에 따라 방향 전환이 올바른지 검증
- 연속 장애물 환경에서 반복 주행 안정성을 확인
- 센서 오차나 노이즈 상황에서도 오동작이 과도하지 않은지 점검

## 기대 학습 성과

- 적외선 센서 기반 인식 시스템 설계 역량 확보
- 자율주행 판단 알고리즘 구현 경험 축적
- 하드웨어와 소프트웨어를 연계한 제어 시스템 이해
- 센서 오차 보정과 성능 검증을 포함한 실무형 프로젝트 수행 능력 강화

## 제안서용 요약 문구

적외선 센서를 이용해 전방 및 주변 장애물을 감지하고, 센서 인식-판단 알고리즘-구동 제어-성능 검증의 4단계 구조로 자율주행 로봇을 구현하는 프로젝트이다. 이를 통해 학습자는 센서 기반 주행 판단과 임베디드 제어, 성능 개선까지 포함한 자율주행 시스템 개발 전 과정을 경험할 수 있다.
